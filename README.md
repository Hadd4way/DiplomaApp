# Diplom App

Diploma project application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Tech stack

- [Vite](https://vitejs.dev/) – build tool
- [React](https://react.dev/) – UI library
